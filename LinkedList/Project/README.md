# Custom Linked List in Java
This project contains **my own implementation of the Linked List Data Structure in Java** without using Java's built-in collections.

## Features
* Add node at head
* Add node at tail
* Insert node at specific index
* Delete node at head
* Delete node using index
* Display linked list
* Track size of the list

## Technologies
* Java
* Object Oriented Programming
* Data Structures

## Purpose
This project was created for **DSA practice and to strengthen my understanding of Linked Lists** while preparing for Java backend development.
