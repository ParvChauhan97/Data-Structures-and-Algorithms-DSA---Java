<h2>📁 File System Explorer (Recursion Based)</h2>

<h4>📌 Project Overview</h4>
File System Explorer is a Java-based mini project that demonstrates the concept of Recursion by traversing directories and files inside a given folder.
The program explores folders recursively and prints the structure of all files and subfolders similar to how a file explorer works.

This project is useful for understanding recursive tree traversal and file handling in Java.
