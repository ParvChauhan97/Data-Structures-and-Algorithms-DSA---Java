<h2>Binary Search in Java</h2>

This repository contains a simple implementation of the Binary Search algorithm in Java.

Binary Search is an efficient searching technique used to find an element in a sorted array.
It works by repeatedly dividing the search interval in half until the target value is found or the interval becomes empty.
