<h2>Student Record Search System (Binary Search)</h2>

<h3>📌 Project Overview</h3>
This project demonstrates how Binary Search can be applied in a real-world scenario.

The program searches for a student record using **Binary Search**, which is much faster than linear search for sorted data.

<h3>🎯 Learning Outcome</h3>
- Understanding Binary Search
- Applying DSA in real-world problems
- Java class and object usage
- Clean code structure for GitHub projects
