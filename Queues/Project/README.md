Ticket Counter Management System (Queue Project)
📌 Project Description:-
This is a simple Java mini project that simulates a Ticket Counter Management System using the Queue Data Structure.

In real-world places like railway stations, banks, hospitals, and service centers, customers are served in First In First Out (FIFO) order.
This project demonstrates how a queue works in such scenarios.

🚀 Functions:- 
--> Add customer to the queue
--> Serve customer from the queue
--> View all waiting customers
--> Check the next customer in line
--> Exit the system
